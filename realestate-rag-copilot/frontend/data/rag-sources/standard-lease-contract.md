# 임대차표준계약서 — 핵심 보호 조항

## 주택임대차표준계약서 (전세·월세 공통)

### 보증금 보호 — 대항력
- 임차인이 **주택 인도 + 전입신고**를 완료한 다음 날부터 제3자에게 대항력 발생.
- 대항력이 있어야 임대인이 바뀌어도(매매·경매) 보증금 반환 청구 가능.
- 매매계약일과 전입신고일이 같은 날이면 새 임대인이 대항력 부정 주장 가능 — **반드시 전입신고가 매매보다 하루 빠르게** 이뤄져야 함.

### 우선변제권 — 확정일자
- 임대차계약서에 **확정일자**를 받으면 경매 시 다른 채권자보다 우선 변제 받을 수 있음.
- 주민센터·인터넷등기소·법원에서 확정일자 부여.
- 비용: 600원 / 1건 (인터넷등기소 기준).

### 소액임차인 최우선변제권 (서울 기준 2026)
- 보증금 1억 6,500만원 이하면 5,500만원까지 다른 채권자보다 무조건 우선 변제.
- 다세대·연립주택은 같은 호수에 여러 임차인 있을 경우 안분 배분.

### 임대차 갱신 청구권 (계약갱신요구권)
- 임차인은 **계약 만료 2개월 전까지** 계약 갱신 요구 가능.
- 임대인은 정당한 사유(본인·직계존비속 실거주 등) 없으면 거절 불가.
- 갱신 시 보증금·월세 인상은 **5% 이내**로 제한.

### 묵시적 갱신
- 임대인이 계약 만료 6개월~2개월 전 사이에 갱신 거절·조건 변경 통지를 하지 않으면 동일 조건으로 자동 갱신.
- 묵시적 갱신 후 임차인은 언제든 해지 통지 가능 (3개월 후 효력).

## 전세사기 예방 — 등기부등본 검증

### 계약 전 반드시 확인
1. **소유자 일치**: 임대인 = 등기부등본 소유자.
2. **근저당권 합계 < 매매시세 70%**: 보증금 + 근저당 합계가 시세의 70% 초과면 위험.
3. **신탁등기 없음**: 신탁등기가 있으면 임대인이 실소유자가 아님. 신탁회사 동의 없는 계약은 무효.
4. **압류·가압류 없음**: 압류·가압류가 있으면 임대인 재산이 묶여 보증금 반환 어려움.
5. **선순위 임차인 없음**: 이미 거주 중인 임차인의 보증금이 신규 보증금과 충돌.

### 계약 직후 즉시 조치
1. **확정일자 받기** — 당일 주민센터 방문 또는 인터넷등기소.
2. **전입신고** — 이사 당일 또는 다음날.
3. **전세보증보험 가입** — HUG (주택도시보증공사) 또는 SGI 서울보증.

## 상가건물임대차표준계약서

### 환산보증금 기준 (서울 기준 2026)
- 환산보증금 = 보증금 + (월세 × 100).
- **9억원 이하**: 상가건물임대차보호법 전면 적용 (5% 인상 제한, 갱신요구권 등).
- **9억원 초과**: 일부 조항만 적용 (대항력은 적용, 5% 제한·갱신요구권 미적용).

### 계약갱신요구권
- 최초 계약 + 갱신 합산 **10년**까지 임차인이 계약 갱신 요구 가능.
- 임대인 거절 가능 사유: 임차인의 차임 연체·심각한 불법 영업·재건축 등.

### 권리금 보호 (상가건물임대차보호법 제10조의4)
- 임차인은 계약 종료 6개월~3개월 전 사이에 **다른 임차인 주선 권리** 보유.
- 임대인이 정당한 사유 없이 새 임차인과 계약을 거부하면 **권리금 회수 방해**로 손해배상 청구 가능.

### 차임증감청구권
- 경제 사정 변동 시 임차인·임대인 모두 차임 증감 청구 가능.
- 증액의 경우 5% 한도, 청구 후 1년 이내 재청구 불가.

## 계약 시 주의 조항

### 특약사항으로 명시해야 할 항목
1. **수선의무 분담**: 보일러·누수 같은 구조적 수선은 임대인 부담, 전구·문고리 같은 소모품은 임차인 부담.
2. **원상회복 범위**: 임차인이 설치한 시설은 원상복구 의무. 단 사전 동의 받은 인테리어는 제외.
3. **명도 시점**: 계약 종료일에 보증금 반환과 동시 인도 (동시이행 항변권).
4. **반려동물·흡연 금지**: 명시하지 않으면 임차인이 자유롭게 결정 가능.

### 분쟁 발생 시 절차
1. **임대차분쟁조정위원회** (대한법률구조공단·LH·서울시 등 운영) — 무료.
2. **민사조정** — 법원에 신청, 양 당사자 합의 시 판결과 동일 효력.
3. **본안소송** — 보증금 반환 청구·임대료 청구 등.
