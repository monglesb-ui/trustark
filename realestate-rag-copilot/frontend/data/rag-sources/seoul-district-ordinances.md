# 서울 자치구 조례 — 영업·생활규제 핵심

## 흡연구역·금연구역 지정 조례

### 서울특별시 간접흡연 피해 방지 조례
- 모든 학교 정문에서 10m 이내 금연구역.
- 어린이집·유치원 시설 경계 10m 이내 금연구역.
- 공중이용시설 출입구·통행로 금연구역.
- 위반 시 과태료 10만원.

### 자치구 추가 금연구역 (강남구·서초구·양천구 등)
- 카페·음식점 외부 테라스도 금연구역으로 지정한 자치구 있음.
- 영업장 내 흡연실 설치는 시설 기준(공기차폐·환기) 충족 필요.

## 자치구별 영업 제한 사례

### 강남구 — 학원 밀집지역 영업 제한
- **대치동 학원가**: 늦은 시간(22시 이후) 청소년 출입 가능 시설(PC방·노래방) 입점 시 자치구 사전 협의 필요.
- **수능 D-100 기간**: 학원가 인근 100m 내 행사·소음 발생 영업 일부 제한.

### 서초구 — 강남대로 상업지구
- **법조타운**(서초역 인근): 1·2종 근린생활시설 비율 40% 제한.
- **고급 일반음식점·바**: 영업시간 02시 제한 (구청 권고).

### 양천구 — 목동 학원·신정동 주거지구
- **목동 1~9단지**: 인근 학교 밀집으로 PC방·노래방 신규 영업 사실상 불가.
- **신정동 주거지구**: 휴게음식점·일반음식점은 제2종 근린생활시설 건물에만 가능.

### 송파구 — 잠실·문정 상업지구
- **잠실 롯데타워 인근**: 야간 영업(00시 이후) 자치구 안전·민원 관리 강화.
- **문정법조단지**: 유흥·단란주점 신규 허가 제한.

## 옥외광고물 조례 (서울특별시 + 자치구 공통)

### 간판 설치 기준
- 1층 영업장 간판: 폭 1m 이내, 높이 60cm 이내.
- 돌출간판: 1면당 2.4㎡ 이내, 보도에서 2.5m 이상 높이.
- 입체간판·전광판: 자치구 사전 허가 필수.

### 위반 시 처분
- 허가 없는 간판: 시정명령 → 강제철거 + 비용 부담.
- 옥외광고물 표시 위반: 과태료 1회 50만원, 2회 100만원.

## 상가건물 영업 제한 (자치구 공통)

### 1종 근린생활시설 (제한적 영업)
- 슈퍼마켓·일용품점 (1,000㎡ 미만).
- 휴게음식점·제과점.
- 의원·치과의원·한의원.
- 미용원·이용원.
- **금지**: 일반음식점, PC방, 노래방, 학원(300㎡ 이상), 단란주점, 유흥주점.

### 2종 근린생활시설 (대부분 영업 가능)
- 일반음식점·휴게음식점·제과점.
- PC방 (인터넷컴퓨터게임시설제공업).
- 노래연습장.
- 학원·교습소 (500㎡ 미만).
- 단란주점 (150㎡ 미만).
- **금지**: 유흥주점 (3종 근린생활시설 또는 일반업무시설 필요).

## 자치구 조례 확인 절차

1. **국가법령정보센터 자치법규** (https://www.law.go.kr/lsSc.do?menuId=14): 자치구명 + 영업 검색.
2. **자치구청 홈페이지**: 정책·조례 메뉴.
3. **위생과·문화체육과 직접 문의**: 영업 가능 여부 사전 컨설팅 (대부분 무료).

## 신규 창업 시 영업 가능성 사전 검증

### 5단계 검증
1. **용도지역 확인** (LURIS): 상업·준주거·일반주거 1·2·3종 여부.
2. **건축물 용도 확인** (BuildingHUB): 1종 근생 / 2종 근생 / 일반업무 / 기타.
3. **자치구 조례 확인**: 해당 자치구 영업 제한 조례.
4. **학교 정화구역 확인**: NEIS + 지도 거리 측정.
5. **임대인 동의 확인**: 표준임대차계약서에 영업 가능 명시.

위 5단계를 모두 통과해야 영업 가능. 한 단계라도 제약이 있으면 사전 협의·조정 필요.
