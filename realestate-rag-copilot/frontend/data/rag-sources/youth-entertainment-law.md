# 풍속영업의 규제에 관한 법률 + 청소년 보호법

## 풍속영업의 범위 (풍속법 제2조)

- 게임산업진흥에 관한 법률에 따른 청소년게임제공업·인터넷컴퓨터게임시설제공업 (**PC방**).
- 음악산업진흥에 관한 법률에 따른 노래연습장업 (**노래방**).
- 식품위생법에 따른 유흥주점·단란주점.
- 사행행위 등 규제 및 처벌 특례법에 따른 사행행위영업.

## PC방 (인터넷컴퓨터게임시설제공업) 영업 기준

### 시설 기준 (게임산업진흥법 시행규칙)
- 1대당 면적 1.5㎡ 이상.
- 환기 시설은 시간당 4회 이상.
- 화재예방 시설: 자동소화장치 + 비상구 표시등.
- 좌석 간격 0.5m 이상 (장시간 이용자 동선 확보).

### 청소년 이용 제한
- **만 16세 미만 청소년**: 오후 10시 ~ 다음날 오전 9시 이용 금지.
- **만 18세 이상 등급 게임**은 18세 미만 이용 절대 금지.
- 신분증 검사 의무 위반 시 영업정지 7일 ~ 1개월.

### 학교환경위생정화구역 (교육환경법 제9조)
- **절대보호구역 (학교 50m)**: PC방 영업 절대 금지.
- **상대보호구역 (학교 200m)**: 교육환경보호위원회 심의 통과 시에만 가능. 통과율 낮음.
- PC방 창업 전 반드시 인근 학교 위치 확인 필수.

## 노래방 (노래연습장업) 영업 기준

### 시설 기준 (음악산업법 시행규칙)
- 객실 면적 합계가 영업장 면적의 50% 이상.
- 객실 출입문은 외부에서 객실 내부를 볼 수 있도록 투명창 설치.
- 객실 안에서 출입문을 잠글 수 없도록 잠금장치 설치 금지.

### 청소년 출입·고용 제한
- 일반 노래방: 청소년 출입 가능 (단, 주류 제공 시 청소년 출입 금지).
- 유흥 노래방·코인 노래방: 청소년 출입 시간 제한 (만 18세 미만은 22시 이후 출입 금지).

### 학교환경위생정화구역
- **절대보호구역**: 노래연습장 영업 금지.
- **상대보호구역**: 심의 통과 시 가능. 일반 노래방은 통과율 높음.

## 청소년 보호법 — 청소년 출입·고용 금지 업소

청소년 (만 19세 미만) 출입·고용이 절대 금지된 업소:
1. 유흥주점·단란주점·룸살롱.
2. 사행행위 영업소 (카지노·경마장 등).
3. 청소년유해매체물·약물·물건 판매·대여업소.
4. 비디오감상실·노래연습장 중 청소년실 비분리 업소.
5. 만화대여업·만화방 (음란성 만화 비치 시).

### 위반 시 처벌
- 청소년 출입 허용: 2년 이하 징역 또는 1천만원 이하 벌금.
- 청소년 고용: 3년 이하 징역 또는 1천5백만원 이하 벌금.
- 신분증 검사 의무 위반: 영업정지 1개월.

## 정화구역 영업 종합 정리

| 업종 | 절대보호구역 (50m) | 상대보호구역 (200m) |
|---|---|---|
| 일반음식점 | 가능 | 가능 |
| 휴게음식점·카페 | 가능 | 가능 |
| 제과점 | 가능 | 가능 |
| PC방 | **금지** | 심의 통과 시 (어려움) |
| 노래방 | **금지** | 심의 통과 시 (가능성 있음) |
| 유흥주점 | **금지** | **금지** (심의 대부분 부결) |
| 단란주점 | **금지** | 심의 통과 시 |
| 만화방 | 가능 | 가능 |
| 미용실·이용원 | 가능 | 가능 |
| 학원·교습소 | 가능 | 가능 |

## 인허가 전 체크리스트 (PC방·노래방 창업자)

1. **학교 위치 확인**: 영업장에서 반경 200m 내 초·중·고·특수학교가 있는지 확인.
   - NEIS 학교알리미 + 다음 지도 거리 측정.
2. **건축물 용도 확인**: 제2종 근린생활시설 이상이어야 PC방·노래방 가능 (1종은 영업 제한).
3. **사전 컨설팅**: 자치구청 위생과·문화체육과에 영업신고 가능 여부 문의 후 임차계약 권장.
4. **임대인 동의**: 임대차계약서에 "PC방·노래방 영업 가능" 명시. 일반 상가 계약서로는 거부 가능.
5. **소방시설 완비증명**: 영업장 면적 100㎡ 이상 자동소화장치·비상구 설치.
